# Stanza — 诗歌结构诊断系统

**Stanza** 是一个完整的现代诗歌结构分析框架，将抽象的审美判断拆解为可执行、可追溯的系统化诊断流程。

## 六引擎架构

```
1️⃣ 无效意象识别    → 逐行定位情绪空转、冗余修辞、功能重复
2️⃣ 意象链结构分析  → 绘制意象流动路径，判断结构类型
3️⃣ 语素精度校准    → 每个实词替换测试，逐词评分 (1-10)
4️⃣ 句法节奏重组    → 句长分布、断行逻辑、节奏弧线
5️⃣ 意象系统图谱    → 意象域提取、关联图绘制、密度分析
6️⃣ 综合评分        → 五维评分 (100分制) + 入选判定
```

## 五维评分体系

| 维度 | 满分 | 评估内容 |
|------|------|---------|
| 意象系统完整性 | 25 | 意象链自洽性、域间缝合精度 |
| 语言控制力 | 20 | 词的不可替换性、冗余度 |
| 主体结构稳定性 | 20 | 主体轨迹清晰度、漂移程度 |
| 诗歌推进与节奏 | 20 | 句法节奏与诗意的匹配度 |
| 原创诗学潜力 | 15 | 不可替代的诗学操作、系统辨识度 |
